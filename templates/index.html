Welcome to Your Automated Content Creator

This app generates and posts short videos based on trends in your niche.

Guides for Usability:

  • Set up API keys in Heroku Config Vars (OPENAI_API_KEY, etc.).
  • Configure NICHE and AFFILIATE_LINK for targeted content.
  • Trigger generation manually below or let the scheduler run daily.
  • View analytics to optimize: Analytics Dashboard

No built-in graphics beyond video generation; outputs are posted directly to platforms.
